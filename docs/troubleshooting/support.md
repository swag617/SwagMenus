# Support

## Before Asking for Help

1. Check the [FAQ](/troubleshooting/faq) — most common issues are covered there
2. Check your server console for error messages on plugin load
3. Make sure you're running **Paper 1.21.x** with **Java 21+**
4. Try `/sm reload` after any config changes

## Getting Help

**Discord** is the fastest way to get support. Join the server and post in the support channel with:

- Your server version (`/version`)
- The error or behavior you're seeing
- The relevant menu YAML file
- Any console errors

[Join Discord](https://discord.gg/z5XZvPQUbG ':class=sf-btn sf-discord')

## Reporting Bugs

Found a bug? Open an issue on GitHub with:

- A clear description of the problem
- Steps to reproduce it
- Your server version and Java version
- The relevant config or YAML snippet
- Console output if applicable

[Open an Issue](https://github.com/swag617/SwagMenus/issues)

## Feature Requests

Feature requests are welcome on GitHub. Check open issues first to avoid duplicates.
