# SwagMenus

SwagMenus is a deeply configurable, player-experience-first GUI menu plugin for Paper servers. Define menus in simple YAML files or build them visually with the built-in web editor — no client mods, no external tools required.

## Highlights

- **YAML-based menus** — one file per menu, auto-reloaded on save
- **Built-in web editor** — full visual GUI builder in your browser
- **15+ action types** — player commands, console commands, sounds, titles, delays, pagination, and more
- **7 requirement types** — gate items and actions behind permissions, money, expressions, and string comparisons
- **PlaceholderAPI support** — use any PAPI placeholder in names, lore, commands, and requirements
- **Pagination** — multi-page menus with next/previous actions
- **Navigation history** — `[back]` returns to the previously opened menu
- **Auto-refresh** — menus update live on a configurable tick interval

## Requirements

| Requirement | Version |
|---|---|
| Paper | 1.21.x |
| Java | 21+ |
| PlaceholderAPI | Any *(optional)* |
