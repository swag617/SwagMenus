# SwagMenus

A Paper plugin for creating fully custom GUIs and menus through YAML configuration. Build menus by editing files directly or use the built-in browser-based visual editor — no client mods or external tools required.

**Requires:** Paper 1.21.x — Java 21+ — PlaceholderAPI *(optional)*

## Download

Head to the [Releases](../../releases) tab to download the latest version.

## Features

- YAML-based menus with hot-reload on save
- Built-in browser-based visual editor
- 15+ action types — commands, sounds, titles, delays, pagination, and more
- 7 requirement types — permissions, money, expressions, placeholders
- PlaceholderAPI support throughout
- Multi-page menus with navigation history
- Fill items, multi-slot items, skull support, hex colors
